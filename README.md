# Images-2.1

Миграция пакета автоматизации системной интеграции на Gulp 4.0. 

Структура проекта:

- gulpfile.js - ядро, содержит основные таски (в дальнейшем будет разбито на подмодули).
- functions.js - вспомогательные функции обработчиков.
- options.js - переменные путей и другие настройки.

Возможности:
1. Оптимизация изображений.
2. Создание спрайтов.
3. Пакетное уменьшение изображений.
4. Переработка legacy-проектов под структуру, принятую в компании.

Использование:
1. git clone https://github.com/Gretten/Images-2.1.git
2. Поместить проект в app/dev
3. Для пересбора legacy-проекта: запустить gulp rebuild для проектов, не требующих замены href в ссылках, или с постфиксом --pre, если замена требуется.
4. Для оптимизации изображений: in process...
